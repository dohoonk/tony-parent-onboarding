# @parent-onboarding/ui

Shared UI components for the Parent Onboarding AI project.

## Purpose

Optional package for UI components that are shared between the web app and potential future applications.

## Usage

```typescript
import { Logo } from '@parent-onboarding/ui'

function MyComponent() {
  return <Logo size="lg" />
}
```

## Components

- **Logo**: Daybreak Health logo component

## Development

Add more shared components as needed. This package is optional and can be expanded over time.
